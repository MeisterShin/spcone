# SPC ONE · 화성시 지능형 의전 운영 플랫폼

행사 준비부터 도착·영접·소개·결과보고까지 실시간으로 연결하는 의전 지휘 플랫폼. 설치 없이 `spc.html` 하나로 동작하는 데모 완성품이며, 향후 React + Firebase 이관 구조는 `SPC_ONE_React_Firebase_이관설계서.md` 참조.

## 실행

`spc.html`을 브라우저에서 바로 열면 된다. 데이터는 브라우저에 저장된다. 웹 배포는 이 폴더 전체를 Netlify 드래그앤드롭 또는 `firebase deploy`(포함된 `firebase.json` 사용).

## 로그인 (역할별 데모 계정)

접속하면 7개 역할 중 선택한다. **서로 다른 탭·기기를 다른 역할로 열면 실시간 협업이 시연된다.**

| 역할 | 권한 |
| --- | --- |
| 시스템 관리자 | 전체 관리·사용자 권한·행사 삭제 |
| 의전 총괄 | 행사·내빈·의전순서·시나리오·영접 전체 |
| 행사 담당자 | 담당 행사 운영·배정·시나리오 확정 |
| 현장 접수자 | 체크인·취소 |
| 영접 담당자 | VVIP 도착 알림·영접 완료 |
| 사회자 | 확정 시나리오·발표 모드(비공개 정보 미노출) |
| 상황실 조회자 | 대시보드·보고 조회 전용 |

## 주요 기능

- **의전 지휘 대시보드** — 실시간 지표, 현장 위험 레이더, 데이터 품질 지킴이, 활동 로그, 연결 상태 표시
- **현장 체크인** — 초성 검색, VVIP·동명이인 확인 단계, 중복클릭 방지, 5초 실행취소, 오프라인 체크인·재연결 자동 동기화
- **VVIP 영접 관리** — 도착 알림, 지연 경보, 영접 담당 지정·완료 처리
- **의전 순서 · 코파일럿** — 규칙 기반 추천 + **추천 근거 표시**, 드래그·▲▼ 수동 조정, 최종 확정은 담당자
- **사회자 시나리오** — 도착 기준 초안 자동 갱신, 확정본 버전관리, 발표 모드(대형 표시·발음 안내), 새 버전 사회자 수락
- **결과보고** — 도착 추이·평균 도착·수정이력·최종 명단 자동 생성, PDF·CSV, 개인정보 최소화 버전

## 개선 반영 (작업지시서 대응)

- **P0 데이터·보안** — 배정 시 기존 체크인 기록 보존(diff 방식), 체크인 취소·실행취소·감사기록, 입력값 이스케이프(XSS 방지), 파일 스키마 검증, 실명 → 가상 인물 교체, 소프트 삭제, 동명이인 탐지, 비공개 메모 노출 차단, CSP 적용
- **P1 핵심** — 역할별 권한, 탭·기기 간 실시간 동기화, 오프라인 큐·멱등 동기화, VVIP 알림·영접, 시나리오 버전관리
- **P2 차별화** — 의전 코파일럿(설명 가능), 데이터 품질 지킴이, 현장 위험 레이더, 발음 정보, 결과보고 자동 생성, 대리참석, 긴급 내빈 추가

## 시연 팁

- 대시보드·체크인 상단의 **오프라인 시연** 버튼으로 인터넷 없이도 체크인→재연결 동기화를 보여줄 수 있다.
- 로그인 화면 우측 하단 **데모 초기화**로 언제든 초기 상태로 되돌린다.
- 인공지능(코파일럿)이 없어도 규칙 기반 추천·시나리오 기본 기능은 정상 작동한다.

## 파일

```
spc.html                                    핵심 앱 (전체 기능)
index.html / _redirects.txt / firebase.json    배포용
SPC_ONE_React_Firebase_이관설계서.md        확장 구조 설계
spc_v4_backup_YYYYMMDD.html                 개선 전 원본 백업
```
